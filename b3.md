# BÀI 3: Đọc hiểu & Dò lỗi qua Prompt (AI Verification)

### 1. Prompt mới do bạn thiết kế
> "Tôi có một hàm Java tính thuế VAT (10% dựa trên `amount`). Tuy nhiên, hàm này đang gặp lỗi logic nghiệp vụ biên khi `amount` là số âm hoặc bằng 0 (`amount <= 0`), vì tiền thuế hoặc giá trị hóa đơn không thể âm. Hãy sửa lại đoạn code Java này để kiểm tra và xử lý chặn lỗi trên bằng cách ném ra ngoại lệ (`IllegalArgumentException`) kèm thông điệp phù hợp."

---

### 2. Đoạn code Java đã được AI sửa đổi

```java
public class TaxCalculator {

    public static double calculateVAT(double amount) {
        // Kiểm tra và xử lý chặn lỗi logic nghiệp vụ biên khi giá trị hóa đơn không hợp lệ
        if (amount <= 0) {
            throw new IllegalArgumentException("Giá trị hóa đơn (amount) phải lớn hơn 0.");
        }
        
        return amount * 0.1;
    }
}
